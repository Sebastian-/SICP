This project aims to provide solutions to every exercise in SICP. 

Many solution sets of this type already exist, so this is more of a personal exercise than anything else. That said, I will try to include explanations with every question for the benefit of anyone referencing this solution set for their own learning.

Solutions are written using the Racket interpreter and its SICP plugin which can be found at:

http://racket-lang.org/

http://www.neilvandyke.org/racket-sicp/

Feel free to contact me with any questions!
